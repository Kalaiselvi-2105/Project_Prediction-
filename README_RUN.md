## Quick Start (Original Working Version)

### 1. Install Dependencies
```bash
cd "AI Prediction"
npm install
```

### 2. Setup Database (Postgres)
```
.env file → DATABASE_URL=postgres://user:pass@localhost:5432/dbname
# Or use local SQLite fallback (projects.db)
npm run db:push
```

### 3. Run Development Server
```bash
npm run dev
```
